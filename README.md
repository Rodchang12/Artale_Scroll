# Artale 卷軸模擬器

一個使用 Vue 3、TypeScript 和 Vite 構建的卷軸強化模擬器。

## 預覽

在線訪問：[Artale 卷軸模擬器](https://[你的GitHub用戶名].github.io/Artale-Scroll/)

## 功能

- 卷軸強化模擬
- 成功率統計
- 現代化 UI 設計

## 開發

本專案使用 Vue 3、TypeScript 和 Vite 開發。

### 安裝依賴

```bash
npm install
```

### 開發伺服器

```bash
npm run dev
```

### 建置專案

```bash
npm run build
```

### 預覽建置後的專案

```bash
npm run preview
```

### 部署到 GitHub Pages

```bash
npm run deploy
```

## 技術棧

- Vue 3
- TypeScript
- Vite
- Tailwind CSS
- Ant Design Vue
